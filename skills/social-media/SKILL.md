# Social Media Sub-Skill

**Role:** Create platform-specific social media content, manage posting strategy, monitor engagement, and handle community interaction. Every post is adapted to the platform's culture, format, and audience.

**Parent:** `marketing-lead` (orchestrator routes here for social media tasks)

---

## Trigger

Activated by the marketing-lead orchestrator when:
- The customer requests social media content for any platform
- A content calendar item involves social media posting
- Engagement monitoring or community management is needed
- A social media performance report is requested

## Platform Specifications

### Twitter/X

| Attribute | Value |
|-----------|-------|
| **Character limit** | 280 (premium: 25,000, but keep under 280 for engagement) |
| **Optimal length** | 70-100 characters (highest engagement) |
| **Tone** | Conversational, witty, direct. Short sentences. |
| **Hashtags** | 1-2 maximum. Research relevance. No hashtag stuffing. |
| **Best posting times** | Weekdays 8-10am and 12-1pm (audience timezone) |
| **Media** | Images increase engagement 150%. Use when available. |
| **Threads** | Use for complex topics. Hook in first tweet. Number tweets (1/5, 2/5...). |

**Twitter/X Writing Rules:**
- Lead with the most interesting or surprising point
- Use line breaks for readability
- Ask questions to drive replies
- Tag relevant accounts sparingly (only when genuinely relevant)
- Never use "link in bio" — include the link directly or skip it

**Example Formats:**
```
[Statement that challenges conventional wisdom]

Here's what actually works:

[3-5 short bullet points]

[CTA or question]
```

```
We just [achievement/milestone].

[1-2 sentences of context]

Thank you to [relevant people] for making it happen.
```

### LinkedIn

| Attribute | Value |
|-----------|-------|
| **Character limit** | 3,000 (post), 120 (headline) |
| **Optimal length** | 150-300 words (enough for value, short enough to read) |
| **Tone** | Professional but human. Thoughtful. First-person works well. |
| **Hashtags** | 3-5 relevant hashtags at the end of the post |
| **Best posting times** | Tue-Thu 8-10am (business hours in audience timezone) |
| **Media** | Carousels (PDFs) get highest reach. Native video second. |
| **Hook** | First 2 lines must be compelling (before "...see more" cutoff) |

**LinkedIn Writing Rules:**
- First 2 lines are everything — they determine click-through
- Use line breaks liberally (one idea per line, single-spaced)
- Open with a hook: bold statement, question, or story opener
- End with a question or CTA to drive comments
- Avoid corporate jargon. Write like a person, not a press release.
- Do NOT use "I'm delighted to announce" or "thrilled to share" — be specific

**Example Formats:**
```
[Bold opening statement or question]

[Short paragraph explaining context]

Here's what I learned:

1. [Insight]
2. [Insight]
3. [Insight]

[Question for the audience]

#hashtag1 #hashtag2 #hashtag3
```

```
[Personal story hook — 1-2 sentences]

[What happened — 2-3 sentences]

[What I learned — the insight]

[How it applies to the reader]

Would love to hear your experience with this.

#hashtag1 #hashtag2 #hashtag3
```

### Instagram

| Attribute | Value |
|-----------|-------|
| **Caption limit** | 2,200 characters |
| **Optimal length** | 125-150 characters (short captions) or 500-1000 (long-form value) |
| **Tone** | Warm, visual, community-oriented. Personality-forward. |
| **Hashtags** | 5-15 relevant hashtags (mix of broad and niche). In first comment or end of caption. |
| **Best posting times** | Mon, Wed, Fri 11am-1pm; Tue, Thu 10am-11am |
| **Media** | REQUIRED — Instagram is visual-first. Suggest image concepts. |
| **Stories** | Casual, behind-the-scenes, polls, questions. Ephemeral. |

**Instagram Writing Rules:**
- The image does the heavy lifting — caption supports, not replaces
- First sentence must hook (appears before "...more" truncation)
- Use emojis sparingly and consistently with brand voice
- Line breaks create visual breathing room
- Include a CTA: "Save this post", "Tag someone who...", "Comment your..."
- Hashtag strategy: 5 broad (100K+ posts) + 5 mid (10K-100K) + 5 niche (<10K)

**Example Formats:**
```
[Hook sentence]

[Value or story — 2-3 sentences]

[CTA]

.
.
.
#hashtags #here
```

### Facebook

| Attribute | Value |
|-----------|-------|
| **Character limit** | 63,206 (but optimal is much shorter) |
| **Optimal length** | 40-80 characters for engagement, up to 250 for value posts |
| **Tone** | Community-oriented, warm, conversational. Like talking to friends. |
| **Hashtags** | 0-2 maximum. Facebook hashtags have minimal impact. |
| **Best posting times** | Weekdays 1-4pm, with Wed being highest engagement |
| **Media** | Video gets highest reach (especially native video). |
| **Groups** | Valuable for community building. Less promotional, more helpful. |

**Facebook Writing Rules:**
- Questions and polls drive the most engagement
- Share value first, promote second (80/20 rule)
- Native video outperforms YouTube links (algorithm preference)
- Tag pages and people only when genuinely relevant
- Use Facebook-specific features: polls, events, stories

## Content Creation Process

### Step 1: Understand the Goal

Before writing a single word, determine:
- **What is the business goal?** (awareness, traffic, engagement, conversion)
- **Who specifically is this for?** (check USER.md TARGET_AUDIENCE)
- **What platform(s)?** (adapt format and tone per platform)
- **What is the CTA?** (every post should ask for ONE thing)

### Step 2: Draft Platform-Specific Content

Create a separate version for each platform. Never copy-paste the same text across platforms.

**Adapting the Same Message:**
```
Original message: "We launched a new feature that helps small businesses save 5 hours per week on invoicing."

Twitter/X: "Small businesses: we just cut your invoicing time by 5 hours/week. Here's how → [link]"

LinkedIn: "We spent 6 months talking to small business owners about their biggest time sink. The answer? Invoicing.

So we built something about it.

Our new [feature] automates the parts of invoicing that eat your week — approvals, follow-ups, reconciliation.

Early users are saving 5+ hours per week.

Here's what that looks like in practice: [link]

#smallbusiness #automation #invoicing"

Instagram: "[Visual: before/after time comparison]
5 hours back in your week.

That's what our new invoicing tool gives small business owners. No more chasing payments. No more manual reconciliation.

More time for the work that actually matters.

Link in bio to try it free."

Facebook: "Know what's eating 5+ hours of your week? Invoicing.

We just launched something to fix that. Small business owners in our beta saved 5 hours per week on average.

What would you do with an extra 5 hours? Tell us in the comments."
```

### Step 3: Review and Quality Check

Before presenting content for approval:

```
SOCIAL MEDIA REVIEW CHECKLIST:
[ ] Is it within the platform's character/format limits?
[ ] Does it match the brand voice from USER.md?
[ ] Is the hook strong enough to stop a scroll?
[ ] Is there a clear, single CTA?
[ ] Are hashtags relevant (not stuffed)?
[ ] Would I engage with this if I saw it in my feed?
[ ] Is any claim verifiable? (no exaggerations)
[ ] Are there typos or grammar issues?
[ ] Is the tone right for this platform?
[ ] Does it complement (not duplicate) recent posts?
```

## Posting Schedule Strategy

### Default Posting Cadence (Adjust Based on Resources)

| Platform | Minimum | Recommended | Maximum |
|----------|---------|-------------|---------|
| Twitter/X | 3x/week | 1x/day | 3x/day |
| LinkedIn | 2x/week | 3-4x/week | 1x/day |
| Instagram | 2x/week | 3-4x/week | 1x/day |
| Facebook | 2x/week | 3-4x/week | 1x/day |

### Time Zone Optimization
- Default to the customer's business timezone
- If audience is global, use the largest audience segment's peak time
- Store the optimal posting times in team-memory once determined

### Content Mix (Per Week)
- **40% Value content**: Tips, how-tos, insights (builds trust)
- **30% Engagement content**: Questions, polls, discussions (builds community)
- **20% Promotional content**: Product features, offers, CTAs (drives conversion)
- **10% Brand personality**: Behind-the-scenes, culture, team stories (builds connection)

Never exceed 20% promotional content. Audiences unfollow brands that only sell.

## Engagement Monitoring

### Reply Strategy
When configured to monitor engagement (requires approval per AGENTS.md for public replies):

| Engagement Type | Response Approach |
|----------------|-------------------|
| **Positive comment** | Thank them genuinely. Do not be generic. Reference what they said. |
| **Question** | Answer directly. If complex, offer to continue in DMs. |
| **Complaint** | Acknowledge, apologize if warranted, take to DMs for resolution. |
| **Spam/trolling** | Ignore. Do not engage. Flag if persistent. |
| **Brand mention** | Like/react. Reply if there is something useful to add. |
| **Share/repost** | Thank them. This is the highest form of engagement. |

### Engagement Metrics to Monitor

Track and store in team-memory weekly:
- **Engagement rate** per platform (engagements / reach)
- **Best performing post** (by engagement, clicks, and conversions)
- **Worst performing post** (diagnose why)
- **Follower growth** (net change)
- **Reply/comment volume** (measure community health)
- **Click-through rate** (if links are included)
- **Share rate** (measure virality potential)

## Hashtag Research

### Research Process
1. Identify 3-5 core topics from USER.md KEY_MESSAGES
2. Search each platform for hashtag volume (use analytics tools via credential-proxy)
3. Build a hashtag library:
   - **Broad** (100K+ posts): High visibility, high competition
   - **Mid** (10K-100K posts): Good visibility, moderate competition
   - **Niche** (<10K posts): Low visibility, but highly targeted
4. Store the hashtag library in team-memory with tag `hashtag-library`
5. Refresh monthly — hashtag relevance changes

### Hashtags to NEVER Use
- Banned or shadowbanned hashtags (platform-specific, changes frequently)
- Hashtags unrelated to the content (hurts algorithm performance)
- Competitor brand names as hashtags (unprofessional)
- Trending hashtags unless the content is genuinely relevant (hijacking trends damages brand)
